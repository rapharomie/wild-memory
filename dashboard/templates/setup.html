Guia de Integração Rápida

Passo 1: Instalar

Instale o Wild Memory via pip ou copie o diretório para seu projeto:

pip install wild-memory
# ou
cp -r wild-memory/ seu_projeto/

Passo 2: Registrar

Inicialize o Wild Memory em seu aplicativo:

from wild_memory import WildMemory

wm = WildMemory(
    supabase_url="sua_url",
    supabase_key="sua_chave"
)
wm.initialize()

Passo 3: Adicionar Link na Barra Lateral

Adicione um link HTML no seu aplicativo:

<a href="/wild-memory/dashboard">🧠 Wild Memory</a>
